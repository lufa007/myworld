vim的使用
1.vim有两种模式：
  插入模式:可直接编辑文本：
      按i键：在光标左侧输入正文
      按o键：在光标下一行输入正文
      按O键：在光标上一行输入正文
      按a键：在光标右侧输入正文
      按I键：在光标所在行开头输入正文
      按A键：在光标所在行末尾输入正文
  命令行模式：输入命令，对文本控制和编辑
2.所有快捷键只在命令行模式下作用
3.文件操作：
   打开文件：命令行输入e 文件地址path
   保存文件：w：保存  w!:强行保存 
             q:退出，没有动过文件 q!:不保存退出，动过文件，后悔了
              wq:保存并退出 ZZ:保存并退出，不需冒号
4.设置背景颜色：输入colo 后按tab键有背景提示
5.设置字体：set guifont=Courier(字体样式)\ New:h10(字体大小)
6.移动光标(不需冒号，按键即可)：
           一、单词：
              w:移至下一个单词的词首
              e:跳至当前或下一个单词的词尾
              b:跳至当前或前一个单词的词首
           二、行内：0：绝对行首 ^:行首的第一个非空白字符 $:绝对行尾
           三、行间：G:文章末尾 3G:第三行  gg:文章开头        +- - - - - -+
               50%：光标跳转到文章的50%位置              H--> |           |
               视野之内：H(头),M(中),L(尾)                    |           |
               翻屏 ctrl:f,b                             M--> |   text    |
7.复制粘贴：命令模式下按v键，进入可视模式                     |           |
  移动光标选择文本，jklh选择整行文本                     L--> |           |
  y 复制 yy:复制整行  p 粘贴  d 剪切                          +- - - - - -+
  dd:删除整行  dg:删除整文 3dd:删除3行
  撤销undo：u   重做已撤销的操作redo：ctrl+r
8.查找：
   正向搜索查找：/查找的文本(查找后按下回车后光标跳转，
   命令行输入set nohlsearch关闭高亮功能,输入set nohl后按tab键会自动补齐)
   如果只是想去掉高亮，直接nohlsearch
   反向搜索查找：？/查找的文本
   单字符查找：f字符 3f字符：从光标开始第三个字符 F：反向查找
   重复查找：正向(;)(只在行间进行) 反向(,)
   按*会光标位置取词
9.替换：%s/替换文本/&/r/gc
10.自动补全：CTRL p
11.语法高亮
12.加密文件：X
13.文本缩进：开始行号，末尾行号>：向右缩进
14.括号匹配：%
15.窗口设置:

	:new 上下分屏, 上面是新建文件. 快捷键 ctrl-w, n
	:split 上下分屏, 将当前工作区复制一份. Ctrl-w , s
	:vsplit 左右分屏。将当前工作区复制一份。快捷键 Ctrl-w, v.
	:vsp filename 左右分割frame
	:sp filename 上下分割frame
	:only 取消分屏。将其他工作区关闭，只保留当前工作区。快捷键 Ctrl-w, o.
	:close, 关闭当前分屏。快捷键 Ctrl-w , c.
	:tabc 关闭当前窗口
	:tabo 关闭所有窗口
	:resize num  例如：:res 5，显示行数调整为5行
	:resize+num 把当前窗口高度增加num行
	:resize-num 把当前窗口高度减少num行

	:vertical resize num 指定当前窗口为num列
	:vertical resize+num 把当前窗口增加num列
	:vertical resize-num 把当前窗口减少num列
	:f file 给窗口重命名

	:n 跳至下一个文件
	:n c 可以直接跳到c文件
	:e# 回到刚才编辑的文件
	:Sex 水平分割当前窗口，并在一个窗口中开启目录浏览器
	:ls 显示当前buffer情况
16.ctrl+G:显示全文总行数以及当前所处位置
17.设置行号：set number 取消行号：set nonumber
18.ctrl+u:向上滚动半屏       ctrl+d:向下滚动半屏
   ctrl+E:向上滚动一行       ctrl+Y:向下滚动一行
   ctrl+F:向下滚动一整屏     ctrl+B:向上滚动一整屏屏
19."\>"表示只匹配单词末尾,"\<"表示只匹配单词开头
